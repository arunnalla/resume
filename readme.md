<div align="center">
  <h2>Resume</h2>
</div>

Generates personal resume using [JSON Resume](https://github.com/jsonresume) and [OnePage theme](https://github.com/ainsleyc/jsonresume-theme-onepage).

## Installation

```bash
npm install
```

## Generate Resume

```bash
npx run generate
```
